# 滋卷  

家常菜之一，方言发音同“滋卷”，有滋有味的卷 :) :)  

## 和面  

| 配料  | 用量  |
| :---: | :---: |
| 面粉  | 300g  |
|  油   |  10g  |
|  盐   | 适量  |
|  水   | 135g  |

1. 将除水以外的所有材料混合均匀，之后边加水边搅拌。  
2. 搅成絮状之后，下手揉成团
3. 密封松弛20分钟，下手揉成光滑面团
4. 视案板大小，将面团分成大小均匀的小剂子，密封饧面两小时，方便擀薄

## 馅料  

|  配料  | 用量  |
| :----: | :---: |
|  韭菜  | 150g  |
|  鸡蛋  |  5个  |
|   盐   | 适量  |
|   油   | 少许  |
| 五香粉 | 适量  |
|  粉条  | 150g  |


## 蒸制  

将面团擀薄后，均匀铺上馅料，卷起，开水上锅小火蒸15分钟。
